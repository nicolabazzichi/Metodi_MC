# Esercizi di Integrazione numerica di PDE

Questa repository è una collezione di esercizi fatti durante la preparazione del modulo "Integrazione di PDE" dell'esame di Metodi Numerici, tenuto da Riccardo Mannella e Francesco Califano.

## Funzionalità
* Capire i punti di forza dei vari algoritmi di integrazione e i rispettivi punti deboli;
* Creare animazioni per comprendere fenomeni come gli shock nelle onde meccanche ma anche il funzionamento di sistemi diffusivi derivanti da equazioni di Fokker-Planck;
* divertirsi il più possibile;

## Requisiti
Prima di iniziare, assicurati di avere installato:
* Python 3.x
* Librerie: `numpy`, `matplotlib`, non credo ci sia bisogno di altro
